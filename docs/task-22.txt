День 22. Ассистент для поддержки пользователей
- Сделайте мини-сервис, который отвечает на вопросы пользователей о вашем продукте
- RAG: отвечает, используя документацию и FAQ
- MCP: интеграция с CRM (или просто JSON с текущими пользователями/тикетами)
Пример: “Почему не работает авторизация?” — ассистент отвечает, учитывая контекст тикета
Результат: ассистент, работающий с поддержкой
Формат: Видео + Код

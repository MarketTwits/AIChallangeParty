Пример потока выполнения:
--------------------------
Запрос: "Сколько будет 25 + 17?"
1. START → agent
   - LLM анализирует запрос
   - Решает вызвать add_numbers(25, 17)
   - Возвращает AIMessage с tool_calls
2. agent → tools (условный переход, т.к. есть tool_calls)
   - Выполняется add_numbers(25, 17)
   - Возвращает ToolMessage с результатом "42"
3. tools → agent (безусловный переход)
   - LLM получает результат инструмента
   - Формирует финальный ответ: "25 + 17 = 42"
   - Возвращает AIMessage без tool_calls
4. agent → END (условный переход, т.к. нет tool_calls)
   - Граф завершается
   - Возвращаем финальный ответ пользователю
